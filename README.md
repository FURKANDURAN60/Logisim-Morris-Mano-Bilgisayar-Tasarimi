# Mano Bilgisayar Simülasyonu

Bu proje, Mano bilgisayarının **Logisim** programında oluşturulmuş bir simülasyonudur. Projede kullanılan **ALU'nun diyagramı**, ek olarak `ALU Diyagramı.txt` dosyasında sunulmuştur.
